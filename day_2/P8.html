Semantic tags provide meaning to the structure of a webpage. It tell what the content is

header: defines the header of a webpage, usually contains the main heading and navigation links
nav: defines the navigation links of a webpage
main: defines the main content of a webpage
section: defines a section of a webpage, usually contains related content
article: defines an independent piece of content, such as a blog post or news article
aside: defines content that is related to the main content, but not essential to it, such as a sidebar or advertisement
footer: defines the footer of a webpage, usually contains copyright information and links to related pages
