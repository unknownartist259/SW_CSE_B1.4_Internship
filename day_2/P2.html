HTML Image Tag - Basic Concepts

This program explains the basic usage of the HTML img tag. It covers source path, alternate text, width, height, and title attributes.

1. Basic Image

The src attribute tells the browser where the image file is located. The alt attribute gives alternative text if the image cannot be displayed.

2. Image with Width and Height

The width and height attributes control the displayed size of the image. They help reserve space on the page before the image fully loads.

3. Image with Title Attribute

The title attribute usually shows a tooltip when the mouse is placed over the image.

4. Local Image Path Example

This example shows how a local image path is written. The image will display only if the file exists in the given folder.

5. Remote Image URL Example

Images can also be loaded from another website using a complete URL.

6. Broken Image Example

This image source is intentionally incorrect. If the image does not load, the browser shows the alternate text.
